## React 源码学习仓库

# 如何使用

```shell
git clone https://github.com/powerjett/react-code.git
cd react-code
npm install
npm run dev
```
# 如何调试

启动项目后，当前项目加载的React源代码就在src/react/packages目录下；可以打debugger以及log，查看代码的运行流程
